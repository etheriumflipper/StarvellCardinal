## 1. Выбор и покупка VPS

### 🌐 Лучший вариант это: 

- [FirstByte](https://firstbyte.ru/) — надежный и недорогой VPS с хорошей поддержкой

### 📊 Минимальные требования для бота

```
✅ Процессор:  1 CPU (2 CPU рекомендуется)
✅ ОЗУ:        1 GB RAM (2 GB рекомендуется)
✅ Диск:       10 GB SSD
✅ ОС:         Ubuntu 22.04 LTS
```

### 🛒 Пошаговая покупка (На примере firstbyte)

1. Перейдите на сайт [FirstByte](https://firstbyte.ru/)
2. Выберите "VPS/VDS"
3. Выберите тариф (Нам подойдёт промо тариф за 75 рублей в месяц)
4. Доменное имя: Любое на ваше усмотрение 
5. Выберите ОС: Ubuntu-24.04-amd64
6. Лицензия на панель управления: Без панели управления
7. Оплатить 

### 📩 Получение данных для подключения

После оплаты перейдите во вкладку товары/услуги, виртуальные сервера 
Выберите ваш сервер и нажмите на "Инструкция", там будут указаны данные для подключения:
- IP-адрес сервера: 123.456.789.0 (пример)
- Пользователь: root
- Пароль: ваш_пароль_от_сервера

## 2. Первое подключение к серверу

### 🪟 Для Windows

#### Способ 1: Через PuTTY (более удобно)

1. **Скачайте PuTTY:**
   - 🔗 https://www.putty.org/
   - Скачайте установщик для Windows

2. **Подключение:**
   - Запустите PuTTY
   - В поле "Host Name" введите IP сервера: `123.456.789.0`
   - Port: `22`
   - Connection type: `SSH`
   - Нажмите "Open"

3. **Вход:**
   - login as: `root`
   - password: введите ваш пароль

#### Способ 2: Через MobaXterm (альтернатива PuTTY)

1. **Скачайте MobaXterm:**
   - 🔗 https://mobaxterm.mobatek.net/download.html
   - Скачайте Home Edition (бесплатную версию)
   - Установите и запустите программу
  
2. **Подключение:**

   - Нажмите "Session" -> "SSH"
   - В поле "Remote host" введите IP сервера: `123.456.789.0`
   - В поле "Specify username" введите: `root`
   - Нажмите "OK"  

1. **Вход:**
   
   - Введите ваш пароль при подключении 

### 🐧 Для Linux/Mac

Откройте терминал и выполните:
```bash
ssh root@123.456.789.0
```

## 3. Базовая настройка сервера

### 🔄 Обновление системы

После первого входа обновите систему:

```bash
# Обновление списка пакетов
apt update

# Обновление установленных пакетов
apt upgrade -y

# Установка базовых утилит
apt install -y curl wget nano htop git screen
```

⏱️ Это может занять немного времени.

## 4. Установка необходимого ПО

### 📦 Установка Git

Git необходим для обновления бота через репозиторий.

```bash
# Установка Git
sudo apt install -y git

# Проверка установки
```

```bash

# Если версия ниже 3.11, устанавливаем:

sudo add-apt-repository ppa:deadsnakes/ppa -y

sudo apt install -y python3.11 python3.11-venv python3.11-dev

# Устанавливаем pip
sudo apt install -y python3-pip
```

### 📝 Установка дополнительных инструментов

```bash

sudo apt install -y screen

# Htop - мониторинг ресурсов
sudo apt install -y htop
```

## 5. Установка и настройка бота

### 📥 Клонирование репозитория

```bash
# Переходим в домашнюю директорию
cd ~

# Клонируем репозиторий
git clone "<REPOSITORY_URL>"

# Переходим в папку с ботом
cd "Starvell-cardinal"
```

### 🔧 Установка зависимостей

```bash
# Создаем виртуальное окружение
python3 -m venv venv

# Активируем виртуальное окружение
source venv/bin/activate

# Устанавливаем зависимости
pip install -r requirements.txt
```

⚠️ **При ошибках установки используйте грубую установку:**

```bash
# Грубая установка зависимостей
pip install -r requirements.txt --break-system-packages

# Если обычная установка не работает
pip install -r requirements.txt --force-reinstall --no-cache-dir

# Или установка с игнорированием зависимостей
pip install -r requirements.txt --ignore-installed
```

✅ После успешной установки вы должны увидеть сообщение о завершении без ошибок.

### 🧪 Запуск

```bash
# Запуск бота для проверки
python3 main.py
```

Если всё работает - вы увидите:
```
✅ Бот успешно запущен!
🟢 Бот запущен
```

Остановите бота: `Ctrl + C`

## 6. Автозапуск бота

### 🖥️ Использование Screen 

Screen позволяет запустить бота в фоновом режиме.

#### Создание screen-сессии:

```bash
# Создаем новую screen-сессию с именем 'starvell'
screen -S starvell

# Запускаем бота
cd ~/Starvell-cardinal
python3 main.py
```

#### Управление screen:

```bash
# Отключиться от сессии (бот продолжит работать)
# Нажмите: Ctrl + A, затем D

# Вернуться к сессии
screen -r starvell

# Посмотреть список сессий
screen -ls

# Завершить сессию (остановит бота)
# Находясь в сессии, нажмите: Ctrl + C, затем выйдите: exit
```

## 8. Решение проблем

### ❌ Бот не запускается

#### Проблема: ModuleNotFoundError

```bash
# Переустановка зависимостей
cd ~/Starvell-cardinal
pip install -r requirements.txt --force-reinstall
```

#### Проблема: Permission denied

```bash
# Дать права на файлы
chmod +x main.py
chmod +x install-sc.sh
```

#### Проблема: Git pull не работает

```bash
# Сброс изменений и получение обновлений
cd ~/Starvell-cardinal
git reset --hard
git pull origin main
```

### 🐛 Проблемы с зависимостями

```bash
# Переустановка Python окружения
cd ~/Starvell-cardinal
rm -rf venv
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 🔌 Проблемы с подключением

#### Проверка интернета на сервере:

```bash
ping -c 4 google.com
curl -I https://starvell.com
```

#### Проверка работы API:

```bash
# Проверка через curl
curl -X GET "https://api.starvell.com/v1/user/me" \
  -H "Cookie: ваш_cookie"
```

### 📝 Просмотр полных логов ошибок

```bash
# Запуск с выводом всех ошибок
cd ~/Starvell-cardinal
python3 main.py 2>&1 | tee error.log
```

### 🔄 Полная переустановка

Если ничего не помогает:

```bash
# Удаление старой версии
rm -rf ~/Starvell-cardinal

# Клонирование заново
cd ~
git clone <REPOSITORY_URL>
cd Starvell-cardinal

# Установка
./install-sc.sh
python3 first_setup.py
```
